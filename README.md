<h1>Dependencies to install</h1>
- Node.js v20.12.2 [https://nodejs.org/dist/v20.12.2/node-v20.12.2-x64.msi]
- run ```npm install```

<h1>Run app</h1>
- run ```npm run dev```

<h1>Build app</h1>
- run ```npm run build```
- move dist folder to server
